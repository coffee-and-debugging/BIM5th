Our Story

This blog is built to teach the Django web framework to BIM 5th-semester students.

This blog, provided by Chitwan College of Technology, teaches the Django web framework to BIM 5th-semester students.

This project, created under the guidance of Unique Adhikari at Chitwan College of Technology, focuses on performing CRUD operations using the Django web framework and MySQL database, providing BIM 5th-semester students with a solid understanding of web development.
